Help

How to use

  1. Tap Choose PDF (or a tool’s Select/Choose button).
  2. Pick your file(s). Everything runs in your browser.
  3. Use the tool accordions on the home page for actions.

Need help? Email parshasparksprintables@gmail.com.

Frequently Asked Questions

Do my PDFs get uploaded to a server?

No. OptiPDF is a static site. All processing happens in your browser (client‑side) — files never leave your device.

Which browsers are supported?

Latest Chrome, Edge, Safari, and Firefox on desktop and mobile. If something doesn’t open on iPhone, try using Safari and ensure pop‑ups are allowed.

Can I use it offline?

Yes — download the repository as a ZIP from GitHub and open index.html locally. Most tools work offline.

How do I report a bug or request a feature?

Open an issue on GitHub Issues or email us.

Privacy — OptiPDF

Privacy

OptiPDF is a client‑side web app. Your PDFs are processed in your browser; files are not uploaded to a server.

  • Hosting is static via GitHub Pages.
  • No user accounts, no tracking beacons.
  • If you email support, your email provider’s policies apply.

If you have confidentiality requirements, you can also download the site and run it offline.
